# STAT395 Final Year Project: Comparing Sampling Methods

This repository contains the code, report, and presentation for my STAT395 project comparing different sampling methods.

## Files
- `STAT 395 CODE (1).sas` – SAS code for sampling simulations and analysis.
- `STAT395 Comparing Sampling Methods f... .pptx` – presentation slides.
- `STAT395 Group C Report (2).docx` – final written report.

## Tools Used
- SAS for statistical analysis
- Microsoft PowerPoint & Word for reporting

## Project Overview
We compared simple random sampling, stratified sampling, and cluster sampling to determine which method yields the most accurate estimates for population parameters.
